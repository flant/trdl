What trdl does

  • trdl implements a secure channel for delivering updates from the Git repository to the user's host.

  • trdl makes sure the update is intact/original.

What trdl cannot do

  • trdl cannot protect you against threats involving physical access to the host where the trdl-client is running.

  • trdl cannot protect you against human errors, e.g., incorrect GPG signature quorum configuration, improper build instructions, and faulty Vault config.
