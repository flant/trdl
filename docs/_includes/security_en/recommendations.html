Our recommendations
  • Use an external authentication provider instead of the Vault root token.
  • Set up an NGINX proxy to switch connections to HTTPS and limit access only to specific Vault endpoints.
  • Run Docker on the same host as Vault; block external access to this host.
  • Do not install any other software on the virtual machine where Vault and the trdl plugin are running.
  • Use the proven methods of protecting the OS/host.
