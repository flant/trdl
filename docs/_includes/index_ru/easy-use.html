Простота использования
Разработчикам
Привычные инструменты и готовый процесс.
  • Конфигурация сборки сводится к выбору базового образа и сборочных инструкций.
  • Релиз, публикация каналов и верификация этих операций кворумом выполняются исключительно через Git.
  • Код, конфигурация и GPG-подписи хранятся в Git -- пользователю не требуются дополнительные инструменты и знания для подтверждения и выполнения релизов и публикации каналов.
  • Продуманный процесс доставки и готовая интеграция (GitHub Actions).
Пользователям
Один инструмент для работы и обновления артефактов ПО.
  • Различные варианты использования и обновления ПО.
  • Обновление версий ПО через выбранные каналы.
  • Верификация обновлений из доверенного TUF-репозитория.
