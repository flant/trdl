trdl-сервер
Отвечает за безопасное наполнение и организацию TUF-репозитория.

  • Контроль соответствия каналов обновлений и релизов.

  • Сборка и публикация релизов проекта.

  • Безопасность репозитория.

trdl-клиент
Отвечает за непрерывную доверенную доставку обновлений через каналы с разными уровнями стабильности.

  • Работа с релизными файлами проекта в рамках каналов стабильности.

  • Доверенная работа с файлами TUF-репозиториев.
