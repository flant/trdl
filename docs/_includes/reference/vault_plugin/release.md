## `/release`

Perform a release

### Perform a release



| Method | Path |
|--------|------|
| `POST` | `/release` |

#### Parameters

* `git_tag` (`string: <required>`) — Git tag.
* `git_username` (`string: <optional>`) — Git username.
* `git_password` (`string: <optional>`) — Git password.

#### Responses

* 200 — OK.
